Part 1. Defining Correspondences

The first correspondence I wanted to build with a picture of Hugh jackman and Wolverine. They have never been seen in a room together and I think they look quite similar.

I used the project point correspondences tool given in the spec and used the Delauney mentioned in the lecture.

For the dissolve fraction and the morph fraction I experimented with several time equations, i ended up settling for the following quadratic:

time = t/frames

frac = s*time**2 + (1 - s)*time

With dissolve and warp s = 0.6, -0.2 respectively. (chosen experimentally)

Part 2. Computing the "Mid-way Face""

To compute the midway face I created a function that finds the weighted average of the shape (based on the warp fraction), then applies the same triangulation to the new shape. Finally for each triangle T_m in the mid-way face I solved for the transformation matrix A s.t. A*T_m = T_i. Using this approach to solve for the midway face requires interpolation; I tried a handful of interpolation functions, including bilinear as was suggested, but ended up using nearest neighbor because itâ€™s the fastest and has good quality.
